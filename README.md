# Eye Tracker Next.js App

This is a simple Next.js application where a pair of eyes follows your mouse cursor.

## How to Run

1.  Open your terminal in this directory (`C:\Temp\eye-tracker`).
2.  Run the development server:
    ```bash
    npm run dev
    ```
3.  Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features
- Interactive eye movement using React hooks.
- Smooth transitions and pupil highlights.
- Styled with Tailwind CSS.
